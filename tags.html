重要なHTMLタグ一覧

このページは、初心者シリーズWeb基礎編「「HTML」」で多用されるタグのいくつかをすぐ参照できるクイックリファレンスページとして作られます。このページの作成を通じて、HTMLのテーブル（table)をtableタグやそれに関連するタグの使い方を学びます。

以下のテーブルに含まれるHTMLタグは全体の一部に過ぎませんが、重要なHTMLについてはここで参照できます。

ドキュメントタグ

HTMLタグ タグ名 目的
html htmlタグ HTML文書であることを宣言する
head headタグ 文書のヘッダ情報を表す
body bosyタグ 文書の本体を表す
title titleタグ 文書にタイトルをつける
meta metaタグ その文書に関する情報を指定する

ブロック要素（block elements)

HTMLタグ タグ名 目的
h1–h6 見出し（heading) この中に見出しを置く　（レベル：１–6)
p 段落(パラグラフ: paragraph) この中に文章を入れる
table 表（テーブル: table) この中にテーブルを置く
tr テーブルの行(table row) この中に表の見出しを置く
th/td> 表の見出し(table header) この表の中に見出しを置く
td 表のデータ(table data) この表の中のセルデータを置く
header ヘッダー(header) 見出しであることを示す
div ディブ(division) 区切る
li リスト(list item) リストを作る
ul アンオーダードリスト（unordered list） 順不同なリストを作る
ol オーダードリスト(ordered list) 順番が決まったリストを作る

インライン要素(inline element)

HTMLタグ タグ名 目的 記述例 画面上の表示結果
em 軽い強調(emphasized) テキストを軽く強調する 技術の成熟(<em>technical sophistication</em>) 技術の成熟（technicalo sophistication)
strong 強調（emphasized) テキストを強調する (<strong>少なくとも10億人</strong>) 少なくとも10億人
a アンカー（anchor) ハイパーリンクにする （<a href="https://learnenough.com/">Learn Enough</a>) Learn Enough
img 画像(image) 画像を配置する (<img src="https;//bit.ly/1MZAFuQ" alt="Michael Hartl">)
code コード(code) コードを表す (<code>strong</code>) strong
span スパン(span) 区切る (<span>モビーディック</span>) モビーディック
